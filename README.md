# W5-CampusClubFinder

## Instruction 
### Project Link 
Link Project: https://campus-club-finder-orpin.vercel.app/ 

### Run local
1. Clone this project
```
git clone https://github.com/vDescent/W5-CampusClubFinder.git
```
2. Go to project directory 
```
cd cd W5-CampusClubFinder
```
3. Install dependencies 
```
npm i
```
4. Run this project local
```
npm run dev
```

## Tech stack used
1. Vite, For Setup and Bundling
2. React, For user interface
3. React Router, for navigation between pages
4. Tailwind, for styling pages and handle responsivity
